The Profit Function: General Form

\[\max_Q \{ \pi(Q) = TR(Q) - C(Q) \} \] Since price is exogenous (determined by the market), \(TR = PQ\): \[\pi(Q) = PQ - C(Q)\]

The Profit Function: Per-Unit Form

Since \(C(Q) = ATC(Q) \cdot Q\), we can rewrite profits as:

\[\pi(Q) = [P - ATC(Q)]Q\]

Setup

When we left our representative firm, we could express the short-run production function (with capacity optimized for \(Q = 2500\)) in terms of the cube-root of the labor input, \(f(L|K=25) = 100*25^{2/3}*L^{1/3}\). The cost function is:

\[C(Q) = 10L(Q) + 20K.\] Substituting \(K=25\), and solving \(f(L|K = 25)\) for \(L\), we have:

\[L(Q) = \frac{Q^3}{625,000,000}\] \[C(Q) = \frac{Q^3}{62,500,000} + 500\]

Note that the firm chose (optimized) its capacity on expecting to sell 2,500 units of output. Suppose that after building its capital input, market conditions yielded the following supply and demand functions:

\[Q_d = 6,000 - 2,000P\] \[Q_s = 10,000*P\]

Solve for the market equilibrium price and quantity.

market <- function(x) {
    supply <- 10000*x[1] - x[2]
    demand <- 6000 - 2000*x[1] - x[2]
    return(c(supply, demand))
}
equilibrium <- nleqslv(c(2,2), market)
equilibrium$x
# In our introduction module, we solved the following example where:
# Q_s = P 
# Q_d = 10 - P 
market <- function(x) {                 # x[1] is price; x[2] is quantity
    supply <- x[1] - x[2]               # nleqslv finds "zeros": P - Q_s = 0
    demand <- 10 - x[1] - x[2]          # 10 - P - Q_d = 0
    return(c(supply, demand))           # Return a vector function
}
equilibrium <- nleqslv(c(2,2), market)  # Solve where both equations equal 0.

Analytical Approach

Run the following code to solve the producer’s problem analytically using Ryacas.

Click here to view the steps for the analytical solution.
  1. Load Ryacas.
  2. Define the profit function.
  3. Derive \(d\pi/dQ\) and simplify.
  4. Solve \(d\pi/dQ = 0\) for \(Q\), remove the variables, and reformat to an ordinary R object.

pi <- ysym('0.5*Q - 500 - Q^3 / 62500000') # Define the profit function as a yac_symbol
dpi   <- deriv(pi, 'Q') |>                 # Take the derivative (and simplify)
  simplify()
qStar <- solve(dpi, 0, 'Q')[1] |>          # Solve for Q (take the first [>0] value)
  y_rmvars()                               # Isolate the solution.

The first-order condition for the profit-maximization problem is:

\[\frac{-3 Q ^{2}}{62500000} + 0.5 = 0\]

This is the condition that sets \(MR = MC\).

The profit-maximizing quantity is:

\[Q^* = \sqrt{\frac{0.3125e8}{3} } \]

\[Q^* \approx 3227.486 \]

Computational Approach

Find the solution to the profit-maximization problem numerically.

Careful! Since our profit function has multiple “critical values” (points where change = 0), getting the correct solution will be sensitive to the initial value (pars option)

library(Rsolnp)
loss <- function(x) {
  - 7*x + (0.05*x^3 - 0.9*x^2 + 10*x + 5)
}
solution <- solnp(
  pars = 1,
  fun = loss,  # Solnp calculates minima so we want to minimize loss 
  LB = 0
)
