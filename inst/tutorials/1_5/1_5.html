Introduction

Basic Idea

Make a hypothesis of what would happen if one of the exogenous determinants changes.

  1. Ceteris paribus
  2. Small changes

Determinants

In the Cobb-Douglas example, the demand function for good \(i\) depends on:

  1. Preferences (the budget-share parameters, \(\alpha\) and \(1 - \alpha\));
  2. Income (\(m\));
  3. The price of good \(i\) (\(p_i\));
  4. The price of another good(s) (\(p_j\)).

We will focus mainly on income and price of good 1 (which is the price of the good for good 1 and the price of another good for good 2).

Types of Hypotheses

  1. Qualitative: Which direction?
  2. Quantitative: How much?

Approaches

  1. Graphical (as long as precise values aren’t important)
  2. Computational
  3. Anayltical

Initial Equilibrium

The following code computes the initial equilibrium (which we derived last time):

library(MicroWithR)
a = 0.25
p1 = 1.5
p2 = 1
m = 100
ux <- function(x)
  x[1] ^ (a) * x[2] ^ (1 - a)
bc <- function(x)
  p1 * x[1] + p2 * x[2]
eq <- solnp2(
  c(1, 1),
  ux,
  ineqfun = bc,
  ineqLB = 0,
  ineqUB = m,
  LB = c(0, 0),
  UB = c(Inf, Inf),
  control = list(trace = 0),
  max = TRUE
)

\[x_1^* = 50/3\]

\[x_2^* = 75\]

The next bit of code graphs the equilibrium:

library(ggplot2)
ggplot() +
  geom_function(
    fun = function(x)
      ux(eq$pars) ^ (1 / (1 - a)) / (x ^ (a / (1 - a))),
    color = 'darkgreen'
  ) +
  geom_function(
    fun = function(x)
      m / p2 - x * p1 / p2,
    color = 'red'
  ) +
  geom_point(aes(x = eq$pars[1], y = eq$pars[2]), color = 'blue') +
  geom_text(
    aes(x = eq$pars[1], y = eq$pars[2]),
    label = paste0(
      'x* = (',
      round(eq$pars[1], 2),
      ', ',
      round(eq$pars[2], 2),
      ') \n u(x*) = ',
      round(ux(eq$pars), 2)
    ),
    hjust = 0,
    vjust = 0
  ) +
  lims(x = c(0, 100), y = c(0, 100)) +
  labs(x = "Good 1", y = "Good 2")

Now let’s see how the equilibrium changes when we change (1) income or (2) price.

Income Change

Computational Approach

To measure the effect computationally, we have to specify a numerical amount for the income change, say $5.

  1. For a consumer with the same preferences (Cobb-Douglass with \(\alpha = 0.25\)), and facing the same prices (\(p_1 = 1.5\); \(p_2 = 1\)), compute the new equilibrium for an increase in income from $100 to $105 (dm = 5). Print the (already-stored) initial and new optimal bundles and the initial and new values of the utility function.
dm = 5
dmEq <-
  solnp2(
    c(1, 1),
    ux,
    ineqfun = bc,
    ineqLB = 0,
    ineqUB = m + dm,
    LB = c(0, 0),
    UB = c(Inf, Inf), 
    control = list(trace = 0),
    max = TRUE
  )
dmEq[c(1, 3)]

\[x_1^{**} = 17.5\]

\[x_2^{**} = 78.75\]

\[\frac{\Delta x_1}{\Delta m} \Bigg|_{x_1 = x_1^*} \approx 0.167\]

Using the old and new equilibrium values for the, we can graph the effect of the change.

Price Change

Let’s analyze the effect of a \(\$0.50\) increase in price, and a \(\$0.50\) decrease in price for good 1.

Computational Approach

  1. For a consumer with the same preferences (Cobb-Douglass with \(\alpha = 0.25\)), and facing the price of good 2 and income (\(p_2 = 1\)), compute the new equilibria for an increase in the price of good 1 from \(\$1.50\) to \(\$2.00\) (dp1 = 0.5).
  2. Print the new optimal bundles.

## Iter: 2 fn: -51.4942  Pars:  16.66667 75.00000

  (p1 + dp1) * x[1] + p2 * x[2]

    c(1, 1),

Appendix: Analytical Approach

Basic approach:

  1. Find the demand functions in general form, without specific parameter values for \(m\), \(\alpha\), \(p_1\), and \(p_2\).
  2. Take the derivative of the demand functions with respect to each variable.

R can actually do this super-fast (by vectorizing the derivatives)!

Solving the first order conditions for \(x_1\) and \(x_2\) in terms of the parameters (the general-form solution from last time), we get the demand functions. The following code derives this result.

library(Ryacas)
u <- ysym('x1^(alpha) * x2^(1-alpha)')
L <- u + ysym('lambda*(m - p1*x1 - p2*x2)')
dL <- deriv(L, c('x1', 'x2', 'lambda'))
MU <- c(
  p1*solve(dL[1], 'lambda'), 
  p2*solve(dL[2], 'lambda')) |>
  y_rmvars()
MRS <- with_value(MU[1]/MU[2], 'p1', p1) |>
  with_value('p2', p2) |> yac_str() |>
  Deriv::Simplify() |> ysym()
D <- solve(c(MRS, dL[[3]]), c(ysym('p1/p2'), 0), c('x1', 'x2')) |> 
  y_rmvars() |>
  simplify()
D[1] <- with_value(D[1], 'x2', D[2]) |> 
  simplify() |> suppressWarnings()
D
## {(alpha*m)/p1, ((1-alpha)*m)/p2}

In a more reader friendly typesetting format using Ryacas::tex() we can write this as: \[x_1(p_1, m) = \frac{\alpha m}{p_{1}}\] \[x_2(p_2, m) = \frac{\left( 1 - \alpha \right) m}{p_{2}}\] Notice that for these preferences the increase in consumption for \(x_1\) is linear in the amount of income (it only depends on the exogenous parameters \(\alpha\) and \(p_1\)).

Run the following code to take the derivative of demand for \(x_1\) and \(x_2\) with respect to income and prices.

(statics <- deriv(D, c('m', 'p1', 'p2')))
## {{           alpha/p1,        (1-alpha)/p2},
##  {    (-alpha*m)/p1^2,                   0},
##  {                  0, (-(1-alpha)*m)/p2^2}}

substituting our parameter values, we have:

\[\nabla x_1(m, p_1, p_2) = \begin{pmatrix} \frac{d}{dm}x_1(m, p_1) \\ \frac{d}{dp_1}x_1(m, p_1) \\ \frac{d}{dp_2}x_1(m, p_1) \end{pmatrix} = \begin{pmatrix} \frac{\alpha }{p_{1}} \\ \frac{ - \alpha m}{p_{1} ^{2}} \\ 0 \end {pmatrix} = \begin{pmatrix} \frac{0.25}{1.50} \\ \frac{-25}{1.50^2} \\ 0 \end{pmatrix} \approx \begin{pmatrix} 0.16667 \\ -11.11 \\ 0 \end{pmatrix}\]

and

\[\nabla x_2(m, p_1, p_2) = \begin{pmatrix} \frac{d}{dm}x_2(m, p_1) \\ \frac{d}{dp_1}x_2(m, p_1) \\ \frac{d}{dp_2}x_2(m, p_1) \end{pmatrix} = \begin{pmatrix} \frac{1 - \alpha }{p_{2}} \\ 0 \\ \frac{ - \left( 1 - \alpha \right) m}{p_{2} ^{2}} \end {pmatrix} = \begin{pmatrix} \frac{0.75}{1} \\ 0 \\ \frac{-75}{1^2} \end{pmatrix} \approx \begin{pmatrix} 0.75 \\ 0 \\ -75 \end{pmatrix}\]
